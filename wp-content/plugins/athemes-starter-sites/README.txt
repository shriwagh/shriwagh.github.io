== Description ==

**This plugin is meant to be used only with the aThemes WordPress themes like [Sydney](http://wordpress.org/themes/sydney/) or [Airi](http://wordpress.org/themes/airi/).**

== Installation ==

Go to Plugins > Add New and search for Sydney Toolbox. Install and activate it.
